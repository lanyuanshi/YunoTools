# YunoTools

一个功能丰富的 Android 工具箱应用，基于原生 Kotlin + Android SDK 开发。

## 功能列表

### 首页
- **短视频去水印** - 支持抖音/快手/小红书/B站等短视频解析去水印
- **随机一言** - 每日一句励志语录

### 图像处理
- **图像压缩** - 选择图片后调节质量滑块实时预览压缩效果，支持保存到相册
- **二维码生成** - 输入文字/链接生成高清二维码，支持保存到相册
- **九宫格切图** - 选择图片后自动居中裁剪为正方形并切分为 3×3 九宫格，支持批量保存

### 媒体处理
- **音频分离** - 从视频中提取音频轨道并保存为 m4a 文件
- **视频剪切** - 可视化选择开始/结束时间，裁剪并保存视频片段

### 更多工具
- **弹幕神器** - 手持弹幕滚动显示
- **答案之书** - 随机给出一个答案
- **横屏时钟** - 横屏显示当前时间

### 个人中心
- **解析历史** - 查看过往的解析记录，点击可直接回放结果
- **设置** - 自定义头像、四主题切换（默认/黑色/粉色/蓝色）

## 技术栈
- **语言**: Kotlin
- **UI**: XML Layout + Material Design 3
- **网络**: Retrofit2 + OkHttp3
- **图片加载**: Glide
- **视频播放**: AndroidX Media3 (ExoPlayer)
- **二维码**: ZXing
- **主题引擎**: 自定义 ThemeApplier 全局动态换色

## 安装
### 方法1: Android Studio
1. 克隆仓库
2. 用 Android Studio 打开项目
3. 同步 Gradle
4. 编译并运行

### 方法2: 命令行
```bash
./gradlew assembleRelease
```

## 配置API
修改 `RetrofitClient.kt` 中的 `BASE_URL` 为你自己的解析API地址。
```kotlin
private const val BASE_URL = "https://your-api-domain.com/"
```
