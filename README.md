# YunoTools

**当前版本：v1.0.93**

安装包下载：[release/YunoTools-v1.0.93.apk](release/YunoTools-v1.0.93.apk)

本版本按最新录屏继续修复“看番”：调整点击剧集后的播放器绑定顺序，先刷新并绑定 PlayerView，再对同一个 ExoPlayer 设置播放源，避免因详情页重建导致播放器短时间卡在 00:00/00:00；增加“正在加载视频”状态，并在首帧/READY 后自动清除；继续保留稀饭同集自动换源、Unicode M3U8 解析与播放器错误显示。

一个功能丰富的 Android 工具箱应用，基于原生 Kotlin + Android SDK 开发。

## 功能列表

### 首页
- **短视频去水印** - 支持抖音/快手/小红书/B站等短视频解析去水印
- **AI聊天** - 接入 OpenRouter，支持模型配置、本地历史记录与智能对话
- **小小漫画** - 漫画阅读器，支持翻页浏览

### 底部导航 & 音乐播放器（v1.0.62+）
- **首页 / 播放 / 个人资料** 三导航布局
- **中间播放按钮**：短按播放/暂停当前歌曲，长按弹出毛玻璃音乐面板
- **本地音乐**：读取手机音频库，显示歌名/歌手/时长，可选择播放
- **在线音乐**：并发搜索歌曲狗 / 歌曲海 / Internet Archive 公开音频源；仅对公开可访问音频直链启用播放与下载，整体搜索使用短超时避免慢源拖垮响应
- **收藏**：预留收藏分类
- **面板状态恢复**：音乐面板关闭后再次打开会恢复上次分类、在线音乐关键词与缓存结果
- **右上角关闭按钮**：音乐面板支持“×”关闭，仅关闭弹窗，不中断当前播放器
- **随机/循环播放** 开关
- **播放中图标旋转动画**
- **统一播放器**：所有音频（本地/在线音乐/内置备用）均走 ExoPlayer
- **系统播放通知**：播放时显示基础通知，辅助 Android 通知栏/灵动岛类区域展示当前音乐
- **无点击水波纹/阴影** 极简交互

### 图像处理
- **图像压缩** - 选择图片后调节质量滑块实时预览压缩效果，支持保存到相册
- **二维码生成** - 按新版卡片式界面生成高清二维码，支持前景/背景色、Logo、保存与分享
- **九宫格切图** - 选择图片后自动居中裁剪为正方形并切分为 3×3 九宫格，支持批量保存

### 媒体处理
- **音频分离** - 从视频中提取音频
- **改视频MD5** - 修改视频文件MD5值
- **视频剪切** - 选择视频后拖动范围或用秒级按钮微调，支持生成并保存剪切后的视频

### 更多工具
- **手持弹幕** - 全屏滚动弹幕，支持自定义文字、颜色与速度
- **以图搜番** - 上传动漫截图识别番剧名称、集数、时间点与相似度
- **看番** - 支持 AGE动漫/稀饭动漫播放源选择，用原生布局展示番剧列表、搜索、详情、剧集、内置播放页、独立历史记录、下载记录和设置
- **动漫搜索** - 动漫番剧搜索
- **订阅管理** - 订阅内容管理
- **横屏时钟** - 全屏横屏显示当前时间
- **图片留存助手** - 主动选择图片或从其他应用分享图片，导入副本后支持预览、保存到相册和分享

### 个人中心
- **解析历史** - 查看过往的解析记录，点击可直接回放结果
- **自定义头像** - 在个人中心直接选择头像，支持 GIF / WebP / 视频动态头像
- **设置** - 主题切换

## 技术栈
- **语言**: Kotlin
- **UI**: XML Layout + Material Design 3
- **网络**: Retrofit2 + OkHttp3
- **图片加载**: Glide
- **音视频播放**: AndroidX Media3 (ExoPlayer)
- **系统播放通知**: AndroidX NotificationCompat + Media3 ExoPlayer
- **二维码**: ZXing
- **主题引擎**: 自定义 ThemeApplier 全局动态换色

## 项目结构
```
app/src/main/java/com/yuno/tools/
├── MainActivity.kt              # 主界面（含底部导航、音乐播放逻辑）
├── YunoApp.kt                   # Application
├── data/
│   ├── ApiService.kt            # API接口
│   ├── RetrofitClient.kt        # 网络客户端
│   ├── VideoParseResult.kt      # 数据模型
│   └── UserSettingsStore.kt     # 用户设置存储
├── ui/
│   ├── video/                   # 视频解析相关
│   ├── image/                   # 图像处理（压缩/二维码/九宫格）
│   ├── media/                   # 媒体处理
│   ├── profile/                 # 个人资料/设置/历史
│   └── tools/                   # 其他工具（漫画阅读器/动漫搜索/看番等）
└── util/
    ├── ThemeApplier.kt          # 全局主题引擎
    ├── MusicSearchHelper.kt     # 在线音乐搜索辅助类（多源并发、公开音频直链）
    ├── ClipboardUtil.kt         # 剪贴板工具
    └── DownloadUtil.kt          # 下载工具
```
