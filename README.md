# YunoTools

**当前版本：v1.0.39**

安装包下载：[`release/YunoTools-v1.0.39.apk`](release/YunoTools-v1.0.39.apk)

本版本新增「爱弥斯」动漫主题：使用用户提供插画作为低透明度沉浸背景，配合粉蓝主色、半透明卡片与轻描边，形成接近哔哩哔哩风格的年轻化主题体验。

一个功能丰富的 Android 工具箱应用，基于原生 Kotlin + Android SDK 开发。

## 功能列表

### 首页
- **短视频去水印** - 支持抖音/快手/小红书/B站等短视频解析去水印
- **随机一言** - 每日一句励志语录

### 图像处理
- **图像压缩** - 选择图片后调节质量滑块实时预览压缩效果，支持保存到相册
- **二维码生成** - 输入文字/链接生成高清二维码，支持保存到相册
- **九宫格切图** - 选择图片后自动居中裁剪为正方形并切分为 3×3 九宫格，支持批量保存

### 媒体处理
- **音频分离** - 从视频中提取音频
- **改视频MD5** - 修改视频文件MD5值
- **视频剪切** - 选择视频后拖动范围或用秒级按钮微调，支持生成并保存剪切后的视频

### 更多工具
- **弹幕神器** - 手持弹幕滚动显示
- **答案之书** - 随机给出一个答案
- **横屏时钟** - 横屏显示当前时间

### 个人中心
- **解析历史** - 查看过往的解析记录，点击可直接回放结果
- **设置** - 自定义头像、五主题切换（默认/黑色/粉色/蓝色/爱弥斯）

## 技术栈
- **语言**: Kotlin
- **UI**: XML Layout + Material Design 3
- **网络**: Retrofit2 + OkHttp3
- **图片加载**: Glide
- **视频播放**: AndroidX Media3 (ExoPlayer)
- **二维码**: ZXing
- **主题引擎**: 自定义 ThemeApplier 全局动态换色

## 配置API

```kotlin
private const val BASE_URL = "https://your-api-domain.com/"
```
